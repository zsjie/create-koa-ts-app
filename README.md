# Create Koa TS App

> A boilerplate to create koa app with typescript


## Table of contents

* [Requirements](#requirements)
* [Usage](#usage)
* [Project structure](#project-structure)
* [License](#license)

## Requirements

* [Nodejs](https://nodejs.org/) 14 or later
* Yarn

## Usage

```bash
npx create-koa-ts-app <project-name>
```

## Project structure

```txt
<project-name>
├── node_modules
├── src
│   ├── @types
│   ├── api
│   │   ├── greeting.ts
│   │   └── index.ts
│   ├── app.ts
│   ├── index.ts
│   └── lib
│       └── route.ts
├── tsconfig.json
├── package.json
├── yarn.lock
├── dev.sh
├── LICENSE
└── README.md
```
